# SELMARJ | Instalações Elétricas

## 1. Descrição do Sistema (Mini-Mundo)
A SELMARJ é uma interface web institucional desenvolvida para uma empresa especializada em serviços elétricos. O projeto foca em transmitir credibilidade e segurança, apresentando serviços de montagem de quadros (QDCs), projetos de baixa tensão e manutenção corretiva. O site funciona como uma vitrine digital, permitindo que potenciais clientes visualizem o portfólio técnico da empresa e entrem em contato direto via integração com WhatsApp.

## 2. Tecnologias e Funcionalidades
### Tecnologias Utilizadas:
* **HTML5:** Estruturação semântica de seções e conteúdos[cite: 30].
* **CSS3:** Estilização personalizada, uso de variáveis de cores, layout em Grid e Flexbox para total responsividade.


### Funcionalidades:
* **Navegação Inteligente:** Menu fixo (Sticky Header) com âncoras para seções da página.
* **Design Responsivo:** Interface adaptada para dispositivos móveis e desktop.
* **Call to Action (CTA):** Botões de conversão direta integrados para orçamento instantâneo.

## 3. Procedimentos de Execução Local
Para visualizar o projeto localmente em sua máquina, siga os passos abaixo:
1.  **Clonagem/Download:** Clone este repositório ou baixe e descompacte o arquivo `.zip`.
2.  **Diretório:** Acesse a pasta raiz do projeto onde se encontra o arquivo `index.html`.
3.  **Execução:** Abra o arquivo `index.html` em qualquer navegador web (Chrome, Firefox, Edge, etc.).
